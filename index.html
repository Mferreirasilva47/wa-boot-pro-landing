Dúvidas Frequentes

Funciona no Firefox ou Edge? ▼
Atualmente a extensão funciona apenas no Google Chrome e no Microsoft Edge (baseado em Chromium). No Edge, o processo de instalação é idêntico.
Precisa deixar o WhatsApp Web aberto? ▼
Sim. O WhatsApp Web deve estar aberto em uma aba do Chrome durante os disparos e a maturação. A extensão injeta código na página do WhatsApp Web para enviar as mensagens.
Qual o intervalo mínimo recomendado entre mensagens? ▼
Recomendamos no mínimo 15 segundos entre mensagens. Para chips novos (ainda maturando), use 30 segundos ou mais para evitar bloqueios temporários.
Meus dados ficam seguros? ▼
Sim. Todos os dados (contatos, campanhas, regras) ficam armazenados localmente no seu Chrome, via chrome.storage. Nenhuma informação é enviada para servidores externos.
Como importar contatos em massa? ▼
Na tela do Disparador, clique em "+ Nova Campanha" e use o campo de importação CSV. O arquivo deve ter as colunas: telefone, nome, empresa. Você também pode colar os números diretamente no campo de texto, um por linha.
